# CHUMS App React Presets
A generic template for creating React apps without the overhead of create-react-app

## Notes
This includes the basics for publishing an app the Chums Intranet
Included modules are:
* Bootstrap 5.3 & Bootstrap Icons, React Bootstrap
* React 19
* Redux & Toolkit

## Instructions
* In GitHub click the "Use this template" button, and "Create a new repository"
* Check out the new repository via editor of choice, the examples are for PHPStorm/JetBrains
* Run ```npm install```
  * npm script preinstall will execute ```npm init``` to prompt for package name, etc. 
* Configure PHPStorm
  * set deployment path, eg, /var/www/intranet.chums.com/apps/{app-name}
  * update index.php 
* Write some code
* Build and publish the app
* Make some user happy with a custom app

## Preferred Layout
* src/index.tsx is the entry point for the app
* Feel free to add or remove structure as needed; some apps might not need all of these.

```
|-- dist/                           // generated by vite
|-- src/
|   |-- api/                        // common api calls
|   |-- app/
|   |   |-- App.tsx
|   |   |-- AppRouter.tsx
|   |   \-- configureStore.ts
|   |   components/                 // common components
|   |-- features/                   // features
|   |   |-- feature-1
|   |   |   |-- api/                // api calls for this feature
|   |   |   |-- components/         // components for this feature
|   |   |   |-- context/            // context providers for this feature
|   |   |   |-- hooks/              // hooks for this feature
|   |   |   |-- slices/             // redux slices for this feature
|   |   |   |-- types/              // type definitions for this feature
|   |   |   \-- utils/              // utilities for this feature
|   |   |-- feature-2
|   |   \-- feature-3
|   |-- hooks/                      // common hooks
|   |-- pages/                      // common pages
|   |-- redux/                      // redux store
|   |-- server/                     // server side rendering and serving
|   |-- types/                      // common types
|   |-- utils/                      // common utils
|   \-- index.tsx
|-- public/                         // any files in public/ will be copied to the /dist/assets/ on build
|-- body.php                        // used for server side rendering
|-- index.php                       // used for server side rendering
\-- other global files as needed
```
